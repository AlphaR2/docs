Magicblock is a powerful and versatile open-source framework designed for building on-chain games on Solana/SVM. 
Whether you're developing a fully on-chain game or looking to integrate Solana into your existing game, Magicblock gives you the tools to abstract away a lot of complexity and let you focus on creating the best experiences for your users. 
Magicblock is MIT-licenced and maintained by Magicblock Labs. 


Features
- Solana.Unity.SDK - entry point for Solana Unity SDK
  - [Solana.Unity.Core](https://github.com/magicblock-labs/Solana.Unity-Core) (utilities to build transactions in unity)
  - [Solana.Unity.Metaplex](https://github.com/magicblock-labs/Solana.Unity.Metaplex) (Metaplex program library)
  - [Solana.Unity.Gum](https://github.com/magicblock-labs/Solana.Unity.Metaplex) (session keys)
- IDL-> C# [generator](https://github.com/magicblock-labs/Solana.Unity.Anchor) (C# client code generator from Anchor programs) 
- [Native WebSockets](https://github.com/magicblock-labs/NativeWebSocket) (cross-platform WebSockets library)
- [SOAR](https://docs.magicblock.gg/Frameworks/SOAR) (Solana On-chain Achievements and Rankings)
- SMS integration (Build your game with the Solana Mobile Stack)

In development
- zkUnity
- ECS
- gas abstraction
